Stage 14 — Computing the First Variation

Let:

$$g(\sigma, t) = \log|\zeta(\sigma+it)| - \log|\zeta(1-\sigma+it)|$$

so that E(σ) = ∫ g² dt. Differentiating under the integral sign:

$$\frac{dE}{d\sigma} = 2\int_{-\infty}^{\infty} g(\sigma,t) \cdot \frac{\partial g}{\partial \sigma}(\sigma,t), dt$$

Computing ∂g/∂σ:

$$\frac{\partial g}{\partial \sigma} = \frac{\partial}{\partial \sigma}\log|\zeta(\sigma+it)| - \frac{\partial}{\partial \sigma}\log|\zeta(1-\sigma+it)|$$

For the first term, since log|ζ(s)| = Re(log ζ(s)):

$$\frac{\partial}{\partial \sigma}\log|\zeta(\sigma+it)| = \text{Re}!\left(\frac{\zeta'}{\zeta}(\sigma+it)\right)$$

For the second term, differentiating with respect to σ at (1−σ+it):

$$\frac{\partial}{\partial \sigma}\log|\zeta(1-\sigma+it)| = -\text{Re}!\left(\frac{\zeta'}{\zeta}(1-\sigma+it)\right)$$

Therefore:

$$\frac{\partial g}{\partial \sigma} = \text{Re}!\left(\frac{\zeta'}{\zeta}(\sigma+it)\right) + \text{Re}!\left(\frac{\zeta'}{\zeta}(1-\sigma+it)\right) = \text{Re}(F(\sigma+it))$$

The first variation is:

$$\boxed{\frac{dE}{d\sigma} = 2\int_{-\infty}^{\infty} g(\sigma,t) \cdot \text{Re}(F(\sigma+it)), dt}$$

At σ = ½: By the functional equation, g(½, t) = 0 identically. Therefore dE/dσ = 0 at σ = ½ — not just in the mean, but because the integrand vanishes pointwise. The critical line is a stationary point of the action for a structural reason, not a cancellation argument.

Setting dE/dσ = 0 off the critical line requires:

$$\int_{-\infty}^{\infty} g(\sigma,t) \cdot \text{Re}(F(\sigma+it)), dt = 0$$

The Euler-Lagrange equation is:

$$\boxed{g(\sigma, t) \cdot \text{Re}(F(\sigma+it)) = 0 \quad \text{pointwise}}$$

which holds either when g = 0 (i.e., σ = ½) or when Re(F) = 0. Since Re(F(s)) = 0 on the zero set of ζ (where the logarithmic derivative has poles), this recovers ζ(ρ) = 0 as the equation of motion — exactly as claimed.
